# Package Request

## Request a New Package

1. Check if the package already exists: [pkgs.pkgforge.qaidvoid.dev](https://pkgs.pkgforge.qaidvoid.dev/)
2. Open an issue: [github.com/pkgforge/soarpkgs/issues/new](https://github.com/pkgforge/soarpkgs/issues/new/choose)

Include in your request:
- Package name and homepage
- Why you want it added
- Link to source code or releases

## Add a Package Yourself

See [Creating Packages](/contributing/creating-packages) for a step-by-step guide.

## Package Criteria

Packages should:
- Have an active upstream project
- Be open source (preferably)
- Not duplicate existing packages
- Have clear licensing
