# Contributing

There are several ways to contribute to PkgForge:

## Add a Package

Write an SBUILD recipe and submit a PR to [soarpkgs](https://github.com/pkgforge/soarpkgs). See [Creating Packages](/contributing/creating-packages) for a step-by-step guide.

## Request a Package

Don't want to write the SBUILD yourself? [Request a package](/contributing/package-request) and the community will help.

## Report Issues

- **Package issues**: [soarpkgs issues](https://github.com/pkgforge/soarpkgs/issues)
- **Soar issues**: [soar issues](https://github.com/pkgforge/soar/issues)
- **Documentation**: [pkgforge-docs issues](https://github.com/pkgforge/pkgforge-docs/issues)

## Community

Join the [Discord](https://discord.gg/djJUs48Zbu) to ask questions, get help, and discuss.
