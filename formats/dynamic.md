# Dynamic Binaries

Dynamically linked binaries.

**`pkg_type`: `dynamic`**

::: warning
Dynamic binaries are for **testing only**. Portable [static binaries](/formats/static) should always be preferred. PkgForge has very few dynamic binaries in its repositories.
:::

Dynamic binaries depend on the host system's shared libraries (glibc, etc.) and may not work across different distributions or library versions. Use the [static](/formats/static) format instead for production packages.
