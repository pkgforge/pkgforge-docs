# Troubleshooting

Common issues and solutions when running portable packages.

## Quick Reference

| Problem | Solution |
|---|---|
| "FUSE not supported" | [Install FUSE](/troubleshooting/fuse) or use `--appimage-extract-and-run` |
| Missing fonts/emojis | [Install font packages](/troubleshooting/fonts) |
| User namespace errors | [Enable kernel namespaces](/troubleshooting/namespaces) |

## Topics

- [FUSE](/troubleshooting/fuse) — filesystem in userspace, required for mounting image-based packages
- [Fonts](/troubleshooting/fonts) — missing characters, emojis, and symbols
- [Namespaces](/troubleshooting/namespaces) — user namespace configuration for sandboxing
